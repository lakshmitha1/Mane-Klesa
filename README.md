# Mane-Kelsa

Mane-Kelsa is a Firebase-based Android application that helps customers find, hire, and rate local workers such as electricians, plumbers, painters, cleaners, and technicians.

## Features

* Worker Registration
* Worker Login
* Customer Booking System
* Worker Availability Status
* Rating and Review System
* Firebase Realtime Database Integration
* Profile Management
* Search Workers by Skill and Location
* Booking Status Tracking

## Technologies Used

* Kotlin
* Android Studio
* Firebase Realtime Database
* XML Layouts
* Jetpack Compose
